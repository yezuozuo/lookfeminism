新疆之旅流水账

2021-08-05

Day0，北京 -> 乌鲁木齐

Day0坐的飞机，Day1的凌晨到了乌鲁木齐，防疫耽误了一些时间，从飞机上带下去一个人不知道发生了什么，摆渡车迟迟不到，酒店来接机，先去做了核酸，非常简陋的地下室有两个防疫人员，做的很快，做完会给一张小条子交给酒店。到了酒店已经2点多，赶紧睡觉。

在降落的时候拍到的照片。

做核酸的地方，当时已经是凌晨2点，之后会在新疆遇到很多这样辛苦的一线工作人员。

Day1，乌鲁木齐 -> 赛里木湖 -> 果子沟大桥 -> 清水河镇【698km】

早上和在北京就联系好的司机联系，说起这个司机，最开始是从小红书上找的，小红书上的司机收了我们2k的押金后说他没时间，然后把我们推给了另外一个司机，结果今天一见面他又带了一个新的司机来，相当于被转手了两次，但实际上宇宙万能守恒定律在默默的运行着，这个司机叫赵哥是非常好的一个人。

从连霍高速上往赛里木湖方向走，中午在服务区吃的饭，4个人每人一盘拌面。

下午到了赛里木湖，坐的区间车。区间车进了景区一个转向眼前的景色形容不出来，那么大一片蓝色在山谷中间，因为没有在靠窗的位置所以没照到好看的照片，但是旁边做了两个男孩子，一直在比划着，我正奇怪的看他们的时候，旁边的阿姨说我说他们是聋哑人，听不到说不出来，但是看他们比划着无声的激动地样子，却让人感到纯粹。

从赛里木湖旁的山坡拍的。

天和水一样蓝。

从赛里木湖出来正好到了傍晚，一看表已经8点多了，但是太阳还在天边挂着，这边的落日发生在北京时间10点。

走到果子沟大桥的时候太阳照在大桥上，山坡上有人在拍婚纱照。

晚上住在清水河镇的民宿里，在附近吃了好吃的手抓羊肉。

Day2，清水河 -> 伊宁（喀赞其） -> 伊昭公路 -> 油菜花田 -> 昭苏【280km】

最开始计划走伊昭公路，结果Day1伊昭公路还在封路，今天醒来居然可以走了。

本来计划要去薰衣草庄园，前一天晚上民宿里的客人说是薰衣草都被收割了，所以改成了到昭苏去看油菜花田。

往伊宁走的路上，同行的朋友突然说自己电脑忘在了民宿里，当时我们已经开出了几十公里，给民宿的老板打电话，以为得我们掉头回去再取一趟，结果老板说他开车把电脑送到伊宁去，清水河镇到伊宁接近80公里，我们以为听错了，结果确认了好几遍真的是这样，一路上遇到的都是这样让人觉得是“骗子”的人。

在喀赞其拍到了好看的玩偶和盘子。

在拿到电脑后吃了好吃的冰激凌，往伊昭公路出发，新疆美丽的风景都在路上。

一起来数羊。

云、蓝天和太阳。

如今依然能回味到那种广阔的美。

路边随处可见的牛羊马。

伊昭公路走到距离伊昭还有几十公里的地方突然封路了，看了下高德和百度，都说得往回返再走6个小时绕路，当时差点儿崩溃了，这样的话得10点之后才能到昭苏了，结果封路的交警说那里有一条小路，走一会儿就到了。

我们沿着不知名的小路一直往前走，一路忐忑，居然真的到了昭苏县城。

去看了万亩的油菜花田。

晚上去吃了大盘鸡，加了野蘑菇，但是不好吃，鸡肉太老了。

Day3，昭苏 -> 夏塔古道 -> 特克斯县【215km】

早上路过了成片的向日葵田，想起了李娟写的那本《遥远的向日葵地》。

我那些可笑的心事，可笑的悲苦，可笑的尊严——好像我活着只是为了将它们无限放大，并想尽办法令它们理直气壮地存在。

我泡沫般活着，还奢望这样的生命能够再长久一些，再有意义一些。

到了眼下，面对与我息息相关的一块田野，我却无话可说，无能为力。

我只好拼命地赞美，赞美种子的成长，赞美大地的丰收。我握住一把沙也赞美，接住一滴水也赞美。我有万千热情，只寻求一个出口。只要一个就够了。可荒野紧闭，旁边的乌伦古河日夜不息。我赞美得嘶声哑气，也安抚不了心虚与恐慌。

我不得安宁。无论生活在多么偏远僻静的地方，我的心都不得安宁。

我最嘈杂，最贪婪。我与眼下这世界格格不入。

眼下世界里，青草顶天而生，爬虫昼追日，夜逐月。风是透明的河流，雨是冰凉的流星。

只有我最简陋，最局促。

我酝酿出一份巨大的悲哀，却流不出几行眼泪。我全面坦露自己的软弱，捶胸顿足，小丑般无理取闹。可万物充耳不闻。

我无数遍讲诉自己的孤独，又讲诉千万人的千万种孤独。越讲越尴尬，独自站在地球上，无法收场。

万亩的向日葵地。

远处我们要去的雪山。

排排队，过马路。

喜欢一朵花。

喜欢一只鹰。

喜欢雪山下的马。

走到雪山还要6个小时，以后不知道还有没有机会上到雪上上。

晚上到下一站特克斯，传说中丘处机的八卦城。在特克斯吃了烧烤，羊排，喝了大乌苏和奶啤。

Day4，特克斯 -> 琼库什台【160km】

早上去做了核酸，吃了土豆馅的包子，喝了喝不下去的奶茶。

早上出去转圈，和路边的小牛玩了会儿。

和小鸡和鸡妈妈玩了会儿。

小城里还有热气球。

路边的壁画。

本来计划今天要先去喀拉峻，结果一路上遇到的人都不推荐去那里，所以我们就直接去琼库什台。一路上的风景依然非常美丽。

这里的小孩子从五六岁的时候就开始骑马了。

结果到了琼库什台停电！！！没一个人有信号，和民宿的老板也联系不到，找了半天才在半山腰上找到了老板。

休整了一会儿下午去骑马，第一次骑马一点都没有害怕，感觉又喜欢上了一项运动。

山上的泉水和雪水，非常凉。

两只小牛在蹭痒痒。

大头贴。

带我们骑马的小孩。

我的马。

可以一直看着的落日。

晚上可以看到星星。

Day5，琼库什台 -> 那拉提草原 -> 那拉提镇【340km】

昨天天气预报说未来几天可能会有雨，早上起来后觉得不太乐观就早早的从琼库什台离开，不然一路都是山路，雨下大了就麻烦了。

结果搁浅在了那拉提，一路上高速的服务区加油站都在建，但是公路牌子上显示的都有加油站。打救援电话转接了n次最后一个民警出警。警察小哥哥拿着柴油的油壶给我拍了张用92号油枪加油的汽油，告诉我不要着急。我能不着急吗，我们用的是95号汽油啊。

民警小哥哥来了之后吐槽了半天，说收费站要投诉他们，因为他们的警察没有车牌，还说每月的油费补贴只有600块，根本不够用，不过还是警察靠谱。

最后我们把收费站投诉了。

草原和云。

又是一朵花。

云。

燕子飞过云端。

在那拉提景区一个小伙子让我们赶紧下山，我以为他年纪不小了，结果问他说才16岁，高中放假来这里打工，不然在家天天被他妈骂，景区管吃管住一天再给他150块钱。

Day6，那拉提 -> 巩乃斯林场 -> 巴音布鲁克【168km】

今天新疆的气氛开始变了。北京昌平出现了疫情，整个新疆管控都开始严了起来，同行的朋友在焦虑要不要回北京，不然有可能就回不去了，结果还是先按行程继续走，看一看再说。

到了巩乃斯林场结果巩乃斯森林公园关门了，然后就直接走国道去巴音布鲁克。

在巴音布鲁克看到了天鹅。

天鹅近照。

九曲十八弯。

我们准备在这里要看日落的，运气好的话可以看到这条河同时有9个太阳。结果到了8点多云开始上来了，狂风闪电，工作人员安排我们赶紧下山，因为没预料到天气会突然变成这样，所以就穿了短袖+防晒服就上山了，体验了一会儿失温，真的太可怕了，想起了白银的马拉松运动员。

Day7，巴音布鲁克 -> 独库公路 -> 焉耆【？km】

今天整个新疆开始下雨，不知道我们的运气好还是不好，把所有的景点都玩完之后才开始下雨，今天很多景点就因为天气原因关闭了。

走独库南段的时候路上都是雾，只能看清前面不到10米。

我们本来计划今天沿着独库公路北上到奎屯，休息一晚上后明天到乌鲁木齐，结果走完南段之后，突然独库封路了，前方发生了塌方。

一脸蒙蔽，找个可以停车的地方开始研究路线，我们现在如果去北边的话，除了独库职能走伊犁方向或者往东走去和静，然后经过吐鲁番到乌鲁木齐，伊犁方向到乌鲁木齐要1000多公里，实在不想再回去了，所以我把奎屯的酒店退掉，定了一个叫焉耆（qi第二声）县的酒店。

从独库往回走，到了巴音布鲁克的路口发现堵了十几公里的车，整个新疆开始查防疫。为了快一些，赵哥开始超车，结果过来了几个民兵把司机的行驶证收走了，说我们超车，到前面的交警再给我们。

结果前面碰到特警说了这个事情后，特警说他们不知道居然有人扣行驶证，他们把民警骂了一顿。不出事的时候履行体验特别好，但是一旦出事了就出现了各种不协调的问题。

终于检查排到我们之后，女交警把赵哥教育了一顿放行了我们。

以为今天就能顺利的住到焉耆，结果我们在焉耆的公安检查站又被拦住了……特警说焉耆县对于外来者都得检查，得把我们的身份证发给公安，然后公安再从系统上查，这么一查就查了2个多小时，期间和焉耆的防疫工作人员聊了很多，他们整个焉耆县就两个防疫人员，现在疫情又严重了，他们得各处跑，期间他儿子还给他打电话问他什么时候回家。

期间有两个重庆的女孩被拦下来了，重庆整个市在当天刚刚升级为中风险地区，她们本来打算从焉耆坐火车回重庆，结果特警和防疫人员不让她们进城，如果进城坐不了火车她们就得被隔离14天，如果不进城的话现在她们去哪焉耆县的工作人员就不管了。

两个女生急的开始哭，最后听了防疫人员的建议去了库尔勒，不知道后来怎么样了。

今天乌鲁木齐发布了暴雨预警，上班的早早下班，地铁也停了，地下车库准备好了防洪，结果雨下的很小，这是郑州事件过去后对于其他地区的影响。

Day8，焉耆 -> 小草湖收费站 -> 乌鲁木齐【？km】

昨天听一起接受检查的司机大哥说小草湖检查站会检查非常长的时间，所以今天早早出发，看地图不太堵，结果到了小草湖还是被拦了下来，把我们带到了停车场，停了很多我们这样的带有外地人的车。

在拦住我们的时候，工作人员把我们的身份证都收走了，什么都没和我们说，恐慌就是这样营造的。

听别人说让我们在车上等着就行，会有人来通知我们，我一听这不对劲啊，什么信息都没有，这不是待人宰割吗，然后赶紧去找工作人员，看到一个负责指挥的小胖子，我问他，他说你赶紧让司机去办手续啊，不然得等到什么时候，后来聊起来他说他已经48小时没睡了。

然后就开始等着办手续，一个年轻的工作人员还可以，办的很快，10分钟就把我们办完了，后来才知道他旁边的看起来50多岁的大哥，一个半小时才办完一个手续，他自己都快崩溃了。

等了接近5个小时，警察才来通知我们，要押送我们回乌鲁木齐。

就这样顶着吐鲁番的大风和雨被警车一路押送到乌鲁木齐。

到了乌鲁木齐把我们带到了一个地图上不显示位置，开始对我们研判，大厅里有这样几个判定结果，一个是放行，比如乌鲁木齐本地的司机，一个是中转，一个是核酸，一个是隔离。

因为我们第二天早上就要坐飞机走，而且不是北京昌平的，所以把我们分到了中转的一列，本来说要把我们带到隔离酒店，第二天再把我们送到机场，结果最后又来了工作人员说是酒店没位置了，然后开车把我们扔到了机场发了身份证就让我们走了，和这个工作人员聊起来他也48小时没睡觉了。

而我们前面几个车就不太幸运了，他们都被隔离了，不知道他们现在怎么样了。工作人员说是管吃管住。

最后幸亏机场的太空舱还有房间，能凑合几个小时。

Last Day

以为会在机场接受严格的检查，结果非常轻松就等待登机了，也没有晚点，非常顺利就回了北京，北京下了飞机也没有防疫检查，终于松了一口气。
