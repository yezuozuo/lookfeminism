一个php编译器的问题

2018-06-23

看一个问题$a=1; $a+$a++ 的值是多少

答案是3，根据下图，先执行$a++,在执行+。

$a + ($a++) = 3

那么$a+$a+$a++呢？是4吗

看下图

答案还是3，执行过程是($a+$a) + ($a++) = 3。

那么$a+$a+$a+$a++呢？是4吗？

答案也不是5，答案是4

(($a+$a)+$a) + ($a++) = 4。
