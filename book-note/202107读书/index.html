2021-08-02

《八〇年夏》

作者：玛格丽特·杜拉斯

评分 ✨✨✨✨

夏天，有大太阳，有三十多度的温度，有蓝得发亮的天空，有风暴，风力很强，持续不断，有疯狂的大海，持续的黑夜里挣扎。

还有说来就来的雷雨，但雷雨过后阳光依旧炙人，但雷雨并不驱散海滩上的愁容。什么再也不能驱散它。夏天来得太晚。

奇怪的是，头脑里还在发生着化学反应，热的外面世界都停止转动的时候，大脑还在转动。

从前世界的无限性是伸手可及的，现在不透明的前途一直在困扰我们脆弱和痛苦的头脑，这也许是极其可悲的失败。

PS：我好像第一次有些看懂意识流了。

《上学记》

作者：何兆武（口述） 、文靖

评分 ✨✨✨✨✨

喜欢的就喜欢，不喜欢的就说不喜欢，没关系的。如果和何先生能做同学，想必很合得来吧，胸无大志，厌恶政治，排斥绝对化和纯粹化，无目的的读书，都是三流的废物。

幸福是圣洁，是目高目远的觉悟，是不断地拷问与扬弃。

金岳霖先生问他为什么要学哲学，王浩答道：“我想解决人生问题。”金岳霖先生接下去又问：“那么你解决了没有？”他说：“还没有。”

问他为什么不写1949之后的事情，他笑了笑。

《桶川跟踪狂杀人事件》

作者：清水洁

评分 ✨✨✨✨✨

读这本书的时候一整天阴天，最后作者说自己的女儿14岁就去世了的时候，情绪到了极点。

任何一个巧合就可能击溃一个人的一生，但仍然后善良的人企盼正义，勇敢的人追寻正义。

《蚂蚁的故事》

作者：博尔特·霍尔多布勒 / 爱德华·威尔逊

评分 ✨✨✨

有关群体协调行动、合作互惠和信号交流的那些社会化属性，最佳的参照却来自非常遥远的远亲——蚂蚁蜜蜂等的真社会性昆虫。这一点人类比起来差的太远了。

《湘西散记》

作者：沈从文

评分 ✨✨✨✨✨

本来想去湘西，跟着这本书的路线走一遍，桃源 -> 泊曾家河 -> 泊兴隆街 -> 过柳林岔 -> 泊缆子湾 -> 过梢子铺长潭 -> 夜泊鸭窠围 -> 泊杨家蛆 -> 横石和九溪 -> 离辰州上行 -> 到泸溪 -> 到凤凰。

不能成行，恰好今天北京大雨，把窗户打开，坐在窗边，雨点扫进窗内，假装做个荒唐的梦。梦到躺在尚有些微余热的泥土上，身贴大地，仰面向天，看尾部闪放宝蓝色光辉的萤火虫匆匆促促飞过头顶。沿河是细碎人语声，蒲扇拍打声，与烟杆剥剥的敲着船舷声。半夜后天空有流星曳了长长的光明下坠。

《教养的迷思》

作者：朱迪斯·哈里斯

评分 ✨✨✨

  1. 从我自身的经验来看赞同父母的作用没有想象的那么大
  2. 但没法量化的结论很难打动我
  3. 联想到看《刑侦日记》里的多重人格，每个人其实都有很多个人格，有的能感知到，有的感知不到
  4. 父母除了生命给子女最大的礼物就是黑暗中仍然能够升起的一束光，父母当以身作则

《云没有回答》

作者：是枝裕和

评分 ✨✨✨✨

山内丰德这个人物，他是加害者，还是受害者？可能他也意识到了他自身的双重性，但他给出的结论不是通常人所选择的回避，而是直接选择了死亡。自己越往下活越觉得自身充满了多重性，往往不愿深究，深究的过程是想抓掉头发的痛苦。

是枝裕和不愧为大师，不仅仅在艺术的表现手法上，能够意识到这种双重性并表现给世人，使世人思考，是大师所为。

《闲话闲说》

作者：阿城

评分 ✨✨✨✨

一是世俗，二是小说。让人说话，能印出来，便是好事。汪曾祺钱钟书张爱玲沈从文多会儿读起来都是惊为天人。

《虚无时代》

作者：彼得·沃森

评分 ✨✨✨✨✨

到目前为止今年读过的最好的一本，味道极其鲜美但老道的大杂烩。

虽然是大杂烩，但是每种味道都特别鲜明，仔细辨别能尝到每种食材的味道，每种味道合起来又有不同的味道。

这本书必须单独写一篇读后感了！

即便所有可能的科学问题都已经得到解决，此时我们仍然会感到，对于有关生活的问题，人们尚未初窥门径。
